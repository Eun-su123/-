# 📚 교사 포트폴리오 관리 시스템

예비교사를 위한 학습 자료 관리 웹 애플리케이션입니다.

## 🎯 주요 기능

### 1단계 (MVP) - 완료 ✅
- **자료 업로드**: 과목, 학년군, 영역, 단원, 차시별로 체계적 분류
- **자료 관리**: 검색, 필터링, 다운로드, 삭제 기능
- **통계 분석**: 과목별, 학년군별 분포 시각화
- **연계 추천**: 키워드 기반 자료 추천

## 🚀 실행 방법

### 필수 패키지 설치
```bash
pip install -r requirements.txt
```

또는 개별 설치:
```bash
pip install streamlit pandas
```

### 애플리케이션 실행
```bash
streamlit run 최종_기말_과제.py
```

브라우저에서 `http://localhost:8501`로 접속하세요!

## 📋 사용 방법

1. **자료 업로드**: 
   - 과목, 학년군, 영역, 단원, 차시 정보 입력
   - 키워드 설정 (쉼표로 구분)
   - 파일 업로드 (PDF, Word, PowerPoint, 한글, 이미지, 동영상, 음성)

2. **자료 검색**:
   - 사이드바에서 과목, 학년군, 영역 필터링
   - 검색어로 제목, 단원, 키워드 검색

3. **통계 확인**:
   - 총 자료 수, 과목 수, 용량 확인
   - 과목별, 학년군별 분포 차트
   - 키워드 분석 및 연계 추천
