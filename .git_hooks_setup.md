# GitHub 실시간 동기화 설정 가이드

## 방법 1: 수동 푸시 (간단)

파일을 수정한 후 다음 명령어를 실행:

```bash
git add .
git commit -m "업데이트 내용"
git push
```

## 방법 2: 자동 푸시 스크립트 사용

`auto_push.py` 스크립트를 실행하면 자동으로 푸시됩니다:

```bash
python auto_push.py
```

## 방법 3: 파일 감시 스크립트 (고급)

파일이 변경되면 자동으로 감지하고 푸시하는 스크립트를 만들 수 있습니다.

## 초기 설정 (한 번만)

```bash
# 1. Git 저장소 초기화
git init

# 2. GitHub 저장소 연결 (본인의 URL로 변경)
git remote add origin https://github.com/사용자명/저장소명.git

# 3. 첫 번째 푸시
git add .
git commit -m "초기 커밋"
git branch -M main
git push -u origin main
```

이후에는 `auto_push.py`를 실행하거나 수동으로 푸시하면 됩니다!
