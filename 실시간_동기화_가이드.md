# 🔄 실시간 파일 동기화 가이드

## 🎯 기능 설명

`acid_alkaline.py` 파일을 수정하면 자동으로:
1. ✅ `최종_기말_과제.py`로 복사
2. 🚀 GitHub에 자동 푸시

## 🚀 사용 방법

### 1단계: GitHub 저장소 설정 (최초 1회만)

터미널에서 다음 명령어 실행:

```bash
# Git 저장소 초기화
git init

# GitHub 저장소 연결 (본인의 URL로 변경!)
git remote add origin https://github.com/사용자명/저장소명.git

# 첫 번째 푸시
git add .
git commit -m "초기 커밋"
git branch -M main
git push -u origin main
```

### 2단계: 실시간 동기화 시작

터미널에서 다음 명령어 실행:

```bash
python auto_sync.py
```

이제 `acid_alkaline.py`를 수정하면 자동으로 동기화됩니다!

## 💡 사용 팁

1. **작업 흐름:**
   - `acid_alkaline.py`에서 작업
   - 자동으로 `최종_기말_과제.py`로 복사됨
   - 자동으로 GitHub에 푸시됨

2. **동기화 중지:**
   - 터미널에서 `Ctrl + C` 누르기

3. **수동 푸시:**
   - 동기화 스크립트 없이 수동으로 푸시하려면:
   ```bash
   python auto_push.py
   ```

## ⚠️ 주의사항

- GitHub remote가 설정되어 있어야 자동 푸시가 작동합니다
- Git이 설치되어 있어야 합니다
- 첫 번째 푸시는 수동으로 해야 합니다 (`git push -u origin main`)

## 🔧 문제 해결

### Git이 인식되지 않을 때
- 새 터미널을 열어보세요
- Git Bash를 사용해보세요

### GitHub 푸시가 안 될 때
- GitHub remote가 설정되어 있는지 확인: `git remote -v`
- Personal Access Token을 사용하고 있는지 확인
